NomNetwork Project

Exploring databases, APIs, and UI in a full-stack application.

Course: Introduction to Databases

Term: Summer 2024

Other contributors: Zhiyan yao, Henry Butler, April Zhao

About the Project

This project brought together realistic data generation, REST API planning, and interactive UI design in Streamlit. It allowed exploration of different user stories and personas, showcasing how functionality needs shift across use cases.

Steps Taken

  • Designed a relational database model
  • Wrote simple and complex CRUD SQL queries
  • Accessed and queried a database from Python and Java
  • Performed conceptual and logical modeling
  • Built a CRUD application with an RDBMS back-end
